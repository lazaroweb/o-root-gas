## Rules (always enforced)

1. **TypeScript strict mode** — `"strict": true` in tsconfig. No `any`.
2. **Bundle limit** — `dist/App.html` must stay under 1.3MB (hard limit 1.5MB).
3. **No routing** — single page only. No React Router.
4. **No localStorage** — GAS iframe blocks it. Use component state.
5. **No client-side fetch()** — all data goes through `callServer()` → GAS server functions.
6. **No dynamic import()** — everything bundled statically.
7. **rootDir must be `./dist`** — never change this in `.clasp.json`.
8. **Never edit `dist/`** — always edit `src/`, then build.

## File structure

```
src/
  index.tsx      — React entry point
  App.tsx        — Root component
  gas-client.ts  — callServer() wrapper
  server.ts      — GAS server functions (doGet, getData, etc.)
  types.ts       — Shared TypeScript interfaces
dist/            — Generated by build (never edit directly)
appsscript.json  — GAS manifest
esbuild.mjs      — Build script
package.json
tsconfig.json
```
